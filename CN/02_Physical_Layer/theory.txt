================================================================================
                    PHYSICAL LAYER
================================================================================

Layer 1 of OSI model. Deals with physical transmission of raw bits.


================================================================================
                    1. PHYSICAL LAYER FUNCTIONS
================================================================================

1. Bit Representation:
   - Encoding bits into signals
   - Voltage levels, light pulses

2. Data Rate:
   - Transmission speed (bits per second)

3. Synchronization:
   - Sender and receiver clock synchronization

4. Line Configuration:
   - Point-to-point or multipoint

5. Physical Topology:
   - Mesh, star, ring, bus

6. Transmission Mode:
   - Simplex, half-duplex, full-duplex


================================================================================
                    2. DATA AND SIGNALS
================================================================================

DATA TYPES:
- Analog: Continuous values (voice, video)
- Digital: Discrete values (0s and 1s)

SIGNAL TYPES:
- Analog Signal: Continuous waveform
- Digital Signal: Discrete voltage levels


SIGNAL CHARACTERISTICS:

1. AMPLITUDE:
   - Height of signal
   - Measured in volts

2. FREQUENCY:
   - Cycles per second
   - Measured in Hertz (Hz)

3. PHASE:
   - Position in cycle
   - Measured in degrees

4. WAVELENGTH:
   - Distance of one cycle
   - λ = c/f (c = speed of light)


BANDWIDTH:
Range of frequencies a medium can transmit.
Higher bandwidth = more data capacity.


================================================================================
                    3. DIGITAL TRANSMISSION
================================================================================

LINE CODING:
Converting digital data to digital signal.


1. UNIPOLAR:
   - One voltage level for 1
   - Zero voltage for 0
   - DC component, synchronization issues

   1: ─────┐     ┌─────┐     ┌─────
           │     │     │     │
   0: ─────┴─────┴─────┴─────┴─────


2. POLAR:

   a) NRZ (Non-Return to Zero):
      - NRZ-L: Low=1, High=0
      - NRZ-I: Invert for 1, no change for 0

   b) RZ (Return to Zero):
      - Signal returns to zero mid-bit
      - Better synchronization

   c) Manchester:
      - Transition in middle of bit
      - Low→High = 1, High→Low = 0
      - Used in Ethernet
      - Self-clocking

   d) Differential Manchester:
      - Transition at middle always
      - Transition at beginning = 0
      - No transition at beginning = 1


3. BIPOLAR:
   - Three voltage levels
   - Alternating for 1s
   - Zero voltage for 0

   AMI (Alternate Mark Inversion):
   1: +V    -V    +V    -V
   0:    0     0


================================================================================
                    4. ANALOG TRANSMISSION
================================================================================

MODULATION:
Changing carrier signal based on data.


1. AMPLITUDE MODULATION (AM):
   - Amplitude varies with data
   - ASK for digital: presence/absence of carrier

       High amplitude = 1
       Low amplitude = 0


2. FREQUENCY MODULATION (FM):
   - Frequency varies with data
   - FSK for digital: different frequencies

       High frequency = 1
       Low frequency = 0


3. PHASE MODULATION (PM):
   - Phase varies with data
   - PSK for digital: phase shifts

       0° = 0, 180° = 1


QUADRATURE AMPLITUDE MODULATION (QAM):
Combines amplitude and phase.
More bits per symbol (16-QAM, 64-QAM, 256-QAM).


================================================================================
                    5. MULTIPLEXING
================================================================================

Combining multiple signals on single channel.


1. FREQUENCY DIVISION MULTIPLEXING (FDM):
   - Different frequency bands for each signal
   - Used in: Radio, TV broadcasting
   - Guard bands prevent interference

   Frequency
       ↑
       │ ┌──┐ ┌──┐ ┌──┐
       │ │S1│ │S2│ │S3│
       └─┴──┴─┴──┴─┴──┴──→


2. TIME DIVISION MULTIPLEXING (TDM):
   - Different time slots for each signal
   - Used in: Digital telephony

   a) Synchronous TDM:
      - Fixed time slots
      - May waste bandwidth

   b) Statistical TDM:
      - Dynamic allocation
      - Only active channels get slots

   Time
       ↑
       │ S1 │ S2 │ S3 │ S1 │ S2 │ S3
       └────┴────┴────┴────┴────┴────→


3. WAVELENGTH DIVISION MULTIPLEXING (WDM):
   - Different light wavelengths
   - Used in: Fiber optic
   - DWDM (Dense WDM) for high capacity


4. CODE DIVISION MULTIPLEXING (CDM):
   - Different codes for each signal
   - All use same frequency and time
   - Used in: CDMA cellular


================================================================================
                    6. TRANSMISSION IMPAIRMENTS
================================================================================

1. ATTENUATION:
   - Signal strength decreases with distance
   - Measured in decibels (dB)
   - Solution: Amplifiers, repeaters


2. DISTORTION:
   - Signal shape changes
   - Different frequencies travel at different speeds


3. NOISE:
   - Unwanted signal added to original

   Types:
   a) Thermal Noise: Random electron movement
   b) Induced Noise: External sources (motors)
   c) Crosstalk: Signal from adjacent wire
   d) Impulse Noise: Spikes (lightning)


SIGNAL-TO-NOISE RATIO (SNR):
SNR = Signal Power / Noise Power
SNR(dB) = 10 log₁₀(Signal/Noise)

Higher SNR = better quality


================================================================================
                    7. DATA RATE LIMITS
================================================================================

NYQUIST THEOREM (Noiseless Channel):
Maximum data rate = 2 × Bandwidth × log₂(L)

Where L = number of signal levels

Example:
Bandwidth = 3000 Hz, 2 levels
Rate = 2 × 3000 × log₂(2) = 6000 bps


SHANNON CAPACITY (Noisy Channel):
C = Bandwidth × log₂(1 + SNR)

Example:
Bandwidth = 3000 Hz, SNR = 3162
C = 3000 × log₂(3163) ≈ 34,860 bps

This is theoretical maximum - cannot be exceeded.


================================================================================
                    8. INTERVIEW QUESTIONS
================================================================================

Q1: What is the function of Physical Layer?
A: Transmits raw bits over physical medium.
   Defines voltages, data rates, connectors.

Q2: What is Manchester encoding?
A: Digital encoding with transition in middle of each bit.
   Low→High = 1, High→Low = 0. Self-clocking.

Q3: Difference between FDM and TDM?
A: FDM: Different frequencies for different signals.
   TDM: Different time slots for different signals.

Q4: What is Shannon's theorem?
A: C = B × log₂(1 + SNR)
   Maximum data rate of noisy channel.

Q5: What is attenuation?
A: Loss of signal strength over distance.
   Overcome by amplifiers or repeaters.
